# Fdf
Ce projet consiste à créer graphiquement la représentation schématique d’un terrain en relief.

### Download and compile
```
git clone https://github.com/almaurgit/Fdf.git Fdf && cd ./Fdf 
make re
```


### Usage
`./fdf [-ch] <fichier>`
